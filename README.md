# bet365Spider
bet365 实时数据获取 比赛数据解析 赔率数据解析


## mitmproxy
- 首先安装mitmproxy
- https://www.mitmproxy.org/


## 运行mitmproxy
- mitmweb -s proxy.py -p 8188
- 浏览器设置代理到 127.0.0.1 端口8188
- 打开bet365网站 即可获取到数据
